Cloudflare KV 图床部署教程
📦 完整部署步骤
1. 准备工作
一个 Cloudflare 账号（免费版即可）

已托管的域名（可选，也可以使用 workers.dev 子域名）

2. 创建 KV 命名空间
登录 Cloudflare Dashboard

进入左侧菜单 "Workers 和 Pages"

点击 "KV" 选项卡

点击 "创建命名空间"

名称填写：IMG_KV（或自定义，需记住）

点击创建

3. 创建 Worker 服务
在 Workers 页面，点击 "创建应用程序" → "创建 Worker"

给 Worker 命名，例如：my-image-host

点击 "编辑代码"

4. 写入 Worker 代码
将kv-tc.js中的完整代码复制到编辑器中，替换默认内容：

6. 绑定 KV 命名空间
在 Worker 编辑页面左侧，点击 "设置" → "变量"

找到 "KV 命名空间绑定" 部分

点击 "添加绑定"

变量名称：IMG_KV（必须与代码中的名称一致）

KV 命名空间：选择你创建的 IMG_KV

点击 "保存"

6. 部署并测试
点击编辑器右上角的 "保存并部署"

部署成功后，你会获得一个 Worker 域名，例如：https://my-image-host.workers.dev

访问该域名即可使用图床

🎯 使用说明
上传图片：点击上传区域或拖拽图片

获取外链：点击图片下方的"复制链接"按钮

管理图片：支持删除、刷新列表

外链使用：复制后的链接可直接在网页、Markdown 中使用

⚙️ 可选优化
绑定自定义域名
在 Workers 页面 → 你的 Worker → "触发器"

添加自定义域名（需先在 DNS 中配置）

调整存储限制
KV 免费版有 1GB 存储限制，如需更大空间可升级套餐。

💡 注意事项
免费版 KV 每日有 1000 次写入限制，适合个人使用

图片以 Base64 格式存储，会占用约 1.33 倍原始大小

建议定期清理不需要的图片

部署完成后，您就拥有了一个完全属于自己的、无服务器图床！
